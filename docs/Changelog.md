# Changelog

## Unreleased

## v0.2.0 - 2026-03-25

### 修复

- 重构 `src/main/window.ts` 的窗口实例管理，使用 `WeakMap` 按窗口角色追踪实例，避免销毁后的窗口被继续引用
- 修复设置窗口默认显示不全、浮窗展开后越界定位和预览模式边缘恢复位置回归
- 修复折叠态展示维度单选逻辑，并精简收缩态默认摘要信息
- 修复吸附态把手主用量显示、沿边拖动交互和数字横向排版问题
- 恢复展开态重置时间，并修正 MCP 月额度时间显示
- 调整收缩态厂商名、百分比和中间列对齐，进一步压缩收缩态宽度
- 统一折叠态与展开态的时间文本强化样式，提升低透明度场景下的可读性

### 变更

- 支持在设置面板中调节浮窗透明度，并将最小透明度从 `50%` 下调至 `10%`
- 为低透明度场景补充浮窗表面层保底透明度，避免折叠态、展开态和吸附态接近不可见
- 收敛智谱品牌展示，并为 Provider 注册中心增加 ID 冲突检测
- 启用 `electron-store` 加密保护敏感 Token
- 启用 Electron sandbox 模式以提升默认安全性

### 发布整理

- 重写 README 体系，根 `README.md` 作为语言入口页，新增 `README.zh-CN.md` 和 `README.en.md`
- 更新项目版本为 `0.2.0`，补齐仓库首页、仓库地址和问题反馈元数据
- 从仓库中移除 `output/playwright/stage5_3` 截图产物与未使用的 `src/renderer/src` 脚手架残留

### 验收

- `npm run typecheck` 通过
- `npm run test` 通过
- `npm run build` 通过

## v0.1.0 - 2026-03-22

### 新增

- 新增 Windows 桌面浮窗主界面，支持折叠态和展开态查看额度
- 新增系统托盘菜单，支持刷新、设置和退出
- 新增设置面板，支持厂商配置管理和刷新频率配置
- 新增浮窗拖拽、置顶、边缘吸附和把手恢复交互
- 新增多维度额度勾选展示逻辑，并持久化保存勾选状态
- 新增 `electron-store` 配置持久化，保存窗口位置、窗口状态、刷新频率和厂商配置
- 新增自动刷新与失败退避逻辑
- 新增智谱 Provider，并完成真实线上额度链路接入
- 新增阿里云百炼 Provider 结构与 UI 接入，当前版本使用 mock 数据展示
- 新增 Windows NSIS 安装包构建能力

### 验收

- 阶段 1 至阶段 5 自动化验收已完成
- 智谱真实额度链路已验证可用
- `v0.1.0` 安装包已成功构建、静默安装并验证可启动

### 说明

- 阿里云百炼真实 API 接入仍待后续补齐
- 发布前建议继续人工复核透明浮窗场景下的 checkbox 点击和边缘把手点击恢复
