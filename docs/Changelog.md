# Changelog

## Unreleased

### 修复

- 重构 `src/main/window.ts` 的窗口实例管理，移除模块级全局窗口引用
- 使用 `WeakMap` 按窗口角色追踪浮窗与设置窗口，避免销毁窗口后继续被强引用持有
- 将窗口拖拽与边缘吸附 IPC 处理改为按需解析当前窗口实例，保持现有行为不变
- 将浮窗透明度最低值从 `50%` 下调至 `10%`，同时更新设置面板滑杆范围与提示文案
- 为低透明度场景补充浮窗表面层保底透明度，避免折叠态、展开态和吸附态接近完全不可见

## v0.1.1 - 2026-03-22

### 变更

- 隐藏设置面板中“添加厂商”的阿里云百炼选项，避免在真实链路未完成时继续新增配置
- 保留 `bailian` Provider、主进程分支和已有配置兼容性，便于后续恢复开发
- 补充发布后收尾整理记录，并更新仓库忽略规则

### 验收

- `npm run typecheck` 已通过
- `v0.1.1` Windows 安装包已重新构建并通过构建验证

## v0.1.0 - 2026-03-22

### 新增

- 新增 Windows 桌面浮窗主界面，支持折叠态和展开态查看额度
- 新增系统托盘菜单，支持刷新、设置和退出
- 新增设置面板，支持厂商配置管理和刷新频率配置
- 新增浮窗拖拽、置顶、边缘吸附和把手恢复交互
- 新增多维度额度勾选展示逻辑，并持久化保存勾选状态
- 新增 `electron-store` 配置持久化，保存窗口位置、窗口状态、刷新频率和厂商配置
- 新增自动刷新与失败退避逻辑
- 新增智谱 CodeGeeX Provider，并完成真实线上额度链路接入
- 新增阿里云百炼 Provider 结构与 UI 接入，当前版本使用 mock 数据展示
- 新增 Windows NSIS 安装包构建能力

### 验收

- 阶段 1 至阶段 5 自动化验收已完成
- 智谱真实额度链路已验证可用
- `v0.1.0` 安装包已成功构建、静默安装并验证可启动

### 说明

- 阿里云百炼真实 API 接入仍待后续补齐
- 发布前建议继续人工复核透明浮窗场景下的 checkbox 点击和边缘把手点击恢复
